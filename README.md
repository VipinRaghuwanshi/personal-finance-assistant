# 💰 Personal Finance Assistant

An intelligent AI-based Personal Finance Assistant built using modern web technologies.
This project helps users manage expenses, get financial advice, and interact with an AI chatbot.
---

## 🚀 Features

* 💬 Chat-based finance interaction
* ➕ Add income & expenses easily
* 📊 Real-time balance tracking
* 📈 Expense insights & analytics
* 📉 Visual charts (Chart.js)
* 📅 Monthly financial reports
* 🤖 AI-powered responses

---

## 🛠️ Tech Stack

* **Frontend:** HTML, CSS, JavaScript
* **Backend:** Node.js, Express
* **AI Integration:** Groq API
* **Charts:** Chart.js

## ⚙️ Setup Instructions

1️⃣ Clone the repository

```
git clone https://github.com/your-username/personal-finance-assistant.git
cd personal-finance-assistant
```

2️⃣ Install dependencies

```
npm install
```

3️⃣ Create `.env` file

```
GROQ_API_KEY=your_api_key_here
```

4️⃣ Run the server

```
node app.js
```

5️⃣ Open frontend

* Open `index.html` in browser

---
